# workflow
一個單頁式的個人生產力網頁應用程式，整合了計時器、待辦清單與筆記功能。設計目標是提供一個無干擾、輕量化且跨裝置的工作環境。
https://progress01.github.io/workflow/

## 主要功能

**1. 雙模式計時器**
* 彈性計時：適合零碎或長時間的連續工作，紀錄累計時間。
* 番茄時鐘：內建 25 分鐘專注 / 5 分鐘休息的循環機制。

**2. 待辦事項清單 (To-Do List)**
* 支援優先級設定 (P1 優先、P2 重要、P3 待辦)。
* 自動排序：依據完成狀態、優先級與日期自動排列。
* 支援插入外部連結。

**3. 永久筆記板**
* 提供純文字輸入空間，適合暫存想法或撰寫日誌。
* 內容會自動儲存，重新整理頁面不會遺失。

**4. 專案倒數計時**
* 設定重要專案的截止日期，系統會自動計算剩餘天數。

## 特色

* **無需伺服器**：所有資料皆使用瀏覽器的「本機儲存 (Local Storage)」技術，資料只會存在您的裝置上，不會上傳至雲端，確保隱私與速度。
* **響應式設計 (RWD)**：介面會根據視窗大小自動調整。電腦版為左右分欄，手機與平板則自動調整為上下垂直排列，方便觸控操作。
* **斷網可用**：網頁載入後，即使沒有網路連線也能正常運作。


## 資料管理

* **自動儲存**：您的任何輸入與計時狀態都會即時儲存於瀏覽器中。
* **清除資料**：若需更換裝置或重置狀態，請點擊頁面底部的「重置所有資料」按鈕。

## 注意事項

* 由於資料儲存在瀏覽器內，若您清除瀏覽器的快取 (Cache) 或 cookies，儲存的筆記與任務將會消失。
* 建議定期將重要的筆記內容備份至其他地方。
